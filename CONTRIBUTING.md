# Contributing

Спасибо за интерес к проекту! Любые улучшения и предложения приветствуются.

## Как помочь

- Сообщить о баге через Issue.
- Предложить улучшение или новую фичу.
- Отправить Pull Request с исправлением или новым функционалом.
- Улучшить документацию и примеры.

## Перед отправкой Issue

- Убедитесь, что похожая Issue уже не создана.
- Описывайте проблему максимально конкретно.
- Приложите шаги для воспроизведения, ожидаемый результат и фактический результат.
- Укажите версию PHP, версию пакета и окружение.

## Перед отправкой Pull Request

1. Создайте ветку от `master`.
2. Опишите в PR:
   - Что изменено.
   - Зачем это нужно.
   - Есть ли breaking changes.
3. При возможности добавьте/обновите тесты.
4. Убедитесь, что код проходит проверки (lint/tests, если настроены).

Отправляя Pull Request, вы соглашаетесь, что ваш код распространяется на условиях лицензии репозитория (MIT).
