# Code of Conduct

Все участники проекта обязуются соблюдать следующие принципы:

- Быть уважительными к другим участникам.
- Не допускать личных нападок, оскорблений, дискриминации.
- Предполагать добрые намерения и решать конфликты через диалог.
- Соблюдать законы и правила площадки (GitHub).

Недопустимы:

- Хейт, харассмент, угрозы.
- Публикация персональных данных без согласия.
- Спам и навязчивая реклама.

Если вы столкнулись с нарушением, свяжитесь с мейнтейнером репозитория через Issues или artem@dubinin.me, указанный в профиле.

Мейнтейнер оставляет за собой право удалять комментарии, закрывать обсуждения и блокировать пользователей, нарушающих данный Code of Conduct.
